流量罗盘 · 使用说明
====================

这是什么
--------
抖音/小红书作品发布前的 AI 自查工具:
- 贴视频链接 → 自动提取文案 → 查违规 + 打流量分
- 或者直接粘文案文字,一样查
- 发布前锁一个"盲预测",发布后回来对账,它会越猜越准

怎么开始(第一次用,3 样都要配)
------------------------------
1. 解压这个包到任意文件夹(路径里最好别带中文)
2. 双击「流量罗盘.exe」
3. 首次打开会看到引导卡,照着配三样:
   ① DeepSeek API Key(必填)—— 去 platform.deepseek.com 注册,创建 API Key,几块钱能用很久
   ② 阿里云百炼 API Key(必填)—— 去 bailian.console.aliyun.com 开通,创建 API-KEY,贴链接提取文案要用
   ③ 登录你自己的抖音 —— 点「登录我的抖音」按钮,会弹出抖音页面,拿手机抖音扫码就行
4. 配完就能用了:把抖音分享链接或文案粘进对话框

隐私说明(重要)
----------------
- 两把钥匙、你的抖音登录、所有记录,全部只存在你自己电脑上这个软件旁边的 _data 文件夹里
- 不会上传给作者,也不会上传给任何人
- 想搬家/备份:把 _data 文件夹拷走就是全部数据;想清空:删掉 _data 即可

常见问题
--------
- 双击没反应/闪退:看 _data\app.log 最后几行,里面写着原因
- 提示"还没登录抖音":点右下角「抖音」小按钮重新扫码
- 提示钥匙错误:检查 key 有没有复制全(sk- 开头一整串)
- 只想查文案不想配百炼和抖音?也行,①配好后直接粘文案文字用
