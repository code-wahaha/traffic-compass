【流量罗盘 · 使用说明】

一、这是什么
发布前体检 + 发布后对账的短视频创作工具：
  1. 违规自查——发布前查违禁词、限流风险，给修改建议
  2. 盲预测——发布前 AI 预测这条的表现区间，预测锁定存档，赖不掉
  3. 发布后对账——满 3 天填实际数据，看预测准不准，越用越准
  4. 复盘教练——基于你的对账记录，回答"为什么没起来、下条怎么调"

二、怎么安装
  1. 把整个「流量罗盘」文件夹解压到任意位置（别放中文太深的路径最好）
  2. 双击文件夹里的 流量罗盘.exe
  3. 双击后请耐心等 10~20 秒，窗口才会弹出（首次最慢）
  4. 系统要求：Windows 10 / 11（64位）

三、首次使用
首次打开会让你填 AI 钥匙（只填一次，存在你自己电脑里，不上传）：
  · DeepSeek API Key（必填）：去 platform.deepseek.com 注册 → API Keys → 创建。
    充 10 块钱能用很久。粘贴时注意别复制错，key 无效会导致审核失败。
  · 阿里云百炼 API Key（选填）：链接提取转文字用，去 bailian.console.aliyun.com 开通。

四、日常用法
  发布前：把视频文案粘进输入框 → 出自查报告 → 可追问改写 →
          定稿后点「锁定盲预测」→ 发布你的视频
  发布后：切到「发布后」页 → 点「我已发布」→ 满 3 天回来填播放/点赞/评论/转发
          → 看对账卡 → 有疑问直接问下面的复盘教练

五、注意事项
  · 粘视频链接自动提取文案：需要本机装有浏览器组件和抖音登录态，
    大多数电脑上暂不可用（会提示你直接粘贴文案，效果一样）
  · 你的全部数据在软件文件夹的 _data 目录里，拷走它 = 完整备份
  · 「发布后」页右上角可导出 Excel 账本 / JSON 备份
  · 删除预测重来是不允许的——盲预测的意义就是锁定，这是特性不是 bug

六、常见问题
  · 双击没反应：等 15~20 秒；还不行就看 _data\app.log 里的报错
  · 审核失败 401：DeepSeek key 无效，检查是否复制完整
  · 窗口空白：确认系统是 Win10/11 且 Edge WebView2 正常（系统自带）
