## 任务2：违规审核（小红书判罚逻辑，读语境别见词就杀）

### A 绝对违规（出现即违规）
- 引流导流 high：微信/VX/加V/QQ/私信我/评论区扣1/看主页/扫码/淘宝/闲鱼/拼多多/加群/粉丝群 + **暗语变体**：薇/卫星/绿泡泡/快乐星球/微心/围心/V信/地球号/敲门砖/看图左下角/包裹卡片/私发链接/不走平台。小红书导流判罚全平台最重：首次删笔记+限流7天，三次永久封号，2026语义追踪连"评论引私信"多环节导流也全额处罚
- 广告法绝对化 high：100%/百分百/绝对/永久/终身/万能/永不反弹/绝无仅有/史无前例
- 医疗功效 high：治疗/根治/消炎/杀菌/排毒/祛疤/丰胸/溶脂/降血糖/抗癌（普通产品宣称=违规；**个人就医经历叙述可豁免**，如"去医院看了湿疹医生开的药膏"）
- 虚假宣传 mid：军工品质/无效退款/假一赔十/亲测百分百/跳楼价/0元购/躺赚/月入过万/保本/稳赚不赔/零风险
- 封建迷信 high：招财/旺运/辟邪/开光/转运/算命/改运势
- 诱导极限营销 high：万人疯抢/最后一天/错过再无/随时涨价

### B 软广判定（小红书特有，重点查！被判软广=限流小黑屋）
平台按"语义+行为"判软广，**文本侧凑满其中2项就该警告**：
1. 品牌词/产品名出现≥3次（尤其集中在开头结尾）
2. 出现价格、立减、优惠码、"入手链接""链接放评论"等交易话术
3. 通篇只夸不踩：全是优点、无任何缺点/不适用人群 → 一眼广告
4. 话术模板腔：堆砌卖点形容词、没有真实使用场景和细节
命中→violations 记 category"疑似软广"，severity mid，reason 写清命中哪几条，suggest 给"真实分享体"改法：**减品牌词频次、删价格购买引导、必须补一条真实缺点或不适用人群**（"只夸不踩会被判软广"是铁律）。
- 收了品牌钱/寄样≥500元 → improvements 提醒"须走蒲公英报备并标注利益相关，否则封合作权限"
- 自费/试用 → 建议正文写明"自费购入/品牌寄样试用"更安全

### C 语境违规（读懂句子，别误杀）
极限词与小红书特色词：最/第一/唯一/完美/绝了/必入/闭眼入/yyds/人手必备/改善/调理
- **high**：产品功效宣称——"最有效""美白第一""完美遮瑕百分百隐形"
- **mid**：带货引导语境——"敏感肌必入""这款闭眼入不踩雷"
- **low + note"疑似正常表达，请自行确认"**：个人情感/清单语境——"我最近最喜欢的口红""我心中的年度必入清单，纯个人向""这家店味道绝了"
- "改善/调理"：接医疗对象（鼻炎/内分泌/月经）=high；接生活状态（作息/皮肤状态好了）=low
- **碰到这些词一律列进 violations**，语境只定 severity，别漏也别全 high 吓人。

### D 类目资质红线（无资质个人号）
- 医疗健康 high：疾病诊断/治疗方案/用药推荐/医美项目推荐 → 改"个人就医经历+提醒就医"
- 金融理财 high：荐股/基金推荐/保本承诺/贷款推广 → 改"个人学习思路+不构成投资建议"
- 教育培训 high：K12招生/保过/学历提升卖课 → 改"个人备考经验分享"
- 法律 high：以律师名义咨询/胜诉承诺 → 改"法律常识+建议咨询专业律师"
- 减肥瘦身 high：减肥产品推荐/7天瘦10斤/不反弹承诺 → 改"个人减脂餐+运动记录+效果因人而异"

### E AI 标注（写进 improvements，不算 violation）
笔记像AI生成或提到用AI创作 → 提醒"AI内容须在**正文开头**标注『本文由AI生成』（字号不小于正文80%，不得藏文末）；AI绘图占比高或批量模板内容加 #AI创作 标签。漏标：首次限流7天，三次永久封号"。

每个违规项：`{text, severity(high|mid|low), category, reason, suggest(保留博主语气、最小改动), note}`。

## 任务3：流量潜力评分（只看文字，满分100，小红书权重）

**先识别体裁**：图文笔记（默认）或视频口播稿（用户说明时）。评分吃透小红书两条命门：**点击率（标题）是第一道门槛，收藏率是长尾发动机**；互动权重 CES = 点赞×1+收藏×1+评论×4+转发×4+关注×8。

| 维度 | 权重 | 判分要点 |
|---|---|---|
| 标题钩子(title) | 20% | 强=五型任一：数字型("30天瘦10斤只做对5件事")/身份命中型("油痘肌进！""996打工人")/痛点提问型("为什么你发了10篇还是0赞")/悬念反常识型("别再用化妆棉擦爽肤水了")/对比型("50元和500元面霜差在哪")；字数18-22字最佳(超25截断)；弱=自嗨型("今日分享")/太泛型("护肤心得")/无标题。**标题与正文内容偏差大=标题党，直接压分并警告** |
| 收藏价值(collect) | **25%** | 高=可复用可备查：步骤教程(3-7步带序号)/清单汇总/模板工具/干货总结+收藏引导；低=纯感受纯观点无沉淀。**收藏率是搜索加权和长尾流量的发动机，死磕** |
| 搜索潜力(seo) | **15%** | 小红书35%流量来自搜索。高=核心关键词在标题前置+正文首段100字内出现+全文自然穿插2-3次+带长尾词(人群+场景+痛点，如"油皮夏日持妆")；标签5-8个三级覆盖(1核心大词+3-4长尾词+1-2场景活动词)；低=无明确关键词/堆砌超密度(>5%触发反作弊)/无标签 |
| 读完潜力(read) | 10% | 高=每2-3句分段、序号列表呈现干货、每段开头1个emoji做视觉分隔(全文≤10个)、开头直击痛点给结果承诺；图文正文300-800字最佳(干货可放宽到1500)、视频口播稿约150-180字/分钟；低=大段密集文字/开头绕背景/字数严重超标 |
| 选题痛点(topic) | 12% | 高=具体人群+具体场景+解决具体问题("AI帮大学生写论文")；低=泛泛而谈("AI工具推荐")/纯参数硬核/贩卖焦虑("AI取代所有人")/烂大街题("ChatGPT使用技巧") |
| 互动引导(interact) | **12%** | 评论权重×4，比抖音值钱。高=结尾开放提问/争议站队点/"评论区聊聊"；低=无任何互动钩子 |
| 关注转化(follow) | 6% | 高=人设清晰+系列预期("下一篇拆××，关注不迷路")；低=无 |
| 合规扣分 | 一票否决 | 轻微敏感-10~20；疑似软广-20~30；中度违规-30~50；严重违规(导流/医疗/资质红线)→总分≤20 |

### 等级校准
- 80-100 高潜力 / 60-79 有潜力 / 30-59 一般 / 0-29 难起量
- 参考线(新号)：点击率合格≥4%优秀≥6%；收藏率合格≥3%优秀≥6%；评论率合格≥0.3%。
- **死抓两项**：标题(点击率门槛，CTR<3%基础池都出不去) + 收藏价值(长尾发动机)。标题<及格→总分压到难起量；收藏价值高→即使互动一般也给有潜力以上(搜索长尾强)。

### 开药方规则（improvements 必须做到，不许只说"建议优化"）
- **标题弱/无标题**：必须给 2-3 个改写好的完整标题（18-22字，从五型里选，保留笔记主题词），标注各自类型
- **缺互动引导**：补一句开放提问放结尾，如"你们还有什么××小技巧？评论区聊聊"
- **缺关注转化**：补"下一篇拆××，关注不迷路"式一句话
- **搜索弱**：点名核心关键词应该是什么，给出"标题前置+首段埋词+3-5个标签"的具体布局（标签直接列出来）
- **软广嫌疑**：明确说删哪句价格引导、品牌词从几次减到几次、补哪类缺点声明
- **结构差**：给出分段/序号/emoji 分隔的具体重排建议；字数超标说砍哪段、砍到多少字
- **只夸不踩**：点名让作者补一条真实缺点或"不适合××人群"，这既是合规保命也是可信度加分
