# 抖音作品表现预测补充资料

## 一、数据采集字段清单

### （1）发布前可采集字段（静态属性）

#### 账号基础维度
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 账号当前粉丝量 | 发布时累计粉丝 | 决定冷启动初始量级；1万粉冷启动约为千粉的3-5倍。核心基准变量 |
| 账号健康分 | 实时健康分(满分100) | 90+完整推荐；70以下初始池收缩30%-80% |
| 账号历史平均完播率 | 近30天均值 | 高于赛道均值的账号冷启动通过率+40% |
| 账号历史平均收藏率 | 近30天均值 | 知识类核心；高收藏账号被标高价值，初始优先级高 |
| 账号垂直度评分 | 内容集中度(0-1) | 0.8以上预测误差可降25% |
| 账号认证状态 | 未认证/实名/职业/蓝V | 认证可信度权重高，审核快 |
| 近7天更新频率 | 近7天作品数 | 断更>7天权重回落，初始流量-20%~40% |

#### 内容基础属性维度
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 视频总时长(秒) | 完整时长 | 决定完播率基准阈值；同时长同赛道赛马 |
| 画面比例 | 9:16/16:9/3:4/4:3 | 9:16完播显著高；横屏公域低22% |
| 内容赛道 | AI科普/职场/生活/娱乐等 | 知识类收藏率是娱乐3-5倍，但播放天花板更低 |
| 标题文案字数 | 标题+文案字符数 | 知识类15-25字点击率最高 |
| 话题标签数量 | 标签总数 | 2-3个精准垂直最佳；堆砌降精准度 |
| 内容信息密度 | 单位时长有效知识点(高/中/低) | 越高收藏/复访越高，长尾上限越高 |

#### 内容质量与合规维度
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 是否真人出镜 | 有无真实人物讲解 | 真人出镜知识类平均完播比纯图文/纯AI高18% |
| 是否含AI生成 | 文案/配音/画面 | 合规标注后进专属池；未标注限流 |
| AI内容是否合规标注 | 勾选官方标签+画面显著标注 | 双标注达标得加权；未标注上限锁在冷启动池 |
| 原创度等级 | 纯原创/二创/搬运 | 纯原创晋级概率是搬运6倍+ |
| 是否配完整字幕 | 全程同步字幕 | 知识类完播+12%，收藏+8% |
| 违禁词风险等级 | 高/中/低 | 高风险不过审/进不了冷启动池 |

#### 发布策略维度
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 发布时段 | 具体时间段 | 活跃高峰初始播放快，冷启动反馈及时 |
| 发布入口 | 抖音首页/西瓜/剪映等 | 中视频须指定入口才计三端收益 |
| 是否加入合集 | 是否进合集 | 提升复访，系列内容总播放高40%+ |
| 发布定位 | 是否加地理位置 | 本地内容同城加权；泛知识全国性加定位反而分散 |

### （2）发布后早期信号字段（1-24小时采集）

#### 基础流量进度
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 发布1小时播放量 | 满1h累计播放 | 冷启动速度直接体现；最核心早期变量 |
| 发布3小时播放量 | 满3h累计播放 | 突破3000大概率进中级池；知识类3h与最终相关性0.78 |
| 发布24小时播放量 | 满24h累计播放 | 第一波结束总量级；破1万有60%+概率最终破5万 |

#### 留存与完播
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 3秒留存率 | ≥3秒用户/曝光 | 冷启动第一考核；低于50%卡冷启动池 |
| 5秒完播率 | ≥5秒/播放 | 高于60%晋级初级概率×3 |
| 15秒完播率 | ≥15秒/播放 | 中长视频中间节点 |
| 整体完播率 | 看完全片/播放 | 晋级基础门槛；高于赛道均值30%可稳定晋级 |
| 平均播放时长(秒) | 平均观看时长 | 比完播更精准的粘性指标 |

#### 深度互动
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 点赞率 | 点赞/播放 | 权重较低；>3%基础线 |
| 评论率 | 评论/播放 | >1%易加权；高质量评论触发二次推流 |
| 收藏率 | 收藏/播放 | **当前权重最高**；>2%大概率进中级以上池 |
| 转发率 | 转发/播放 | >0.5%触发破圈裂变 |

#### 粉丝与负向
| 字段 | 含义 | 预测重要性 |
|------|------|-----------|
| 关注转化率 | 新增粉/播放 | 高则判定高价值内容 |
| 铁粉播放占比 | 铁粉播放/总播放 | 2026核心权重；高则基础盘稳 |
| 2秒跳出率 | <2秒划走/曝光 | 负向之王；>40%基本卡冷启动池 |
| 不感兴趣率 | 点不感兴趣/播放 | >1%推流快速衰减 |

## 二、不同粉丝量级 + 赛道 播放量基准参考
（2025-2026 行业经验值，浮动 ±30%；重点标 AI 知识科普类）

### （1）0-1000粉（新号冷启动期）
| 赛道 | 冷启动(1-3h) | 正常作品最终 | 优质作品最终 | 爆款门槛 |
|------|--------------|--------------|--------------|----------|
| 知识科普类(AI/科技) | 200-800 | 500-3000 | 3000-10000 | 1万+ |
| 泛娱乐类(剧情/搞笑) | 300-1000 | 800-5000 | 5000-20000 | 2万+ |
| 生活技能类(美食/家居) | 250-900 | 600-4000 | 4000-15000 | 1.5万+ |

补充：新号默认进种子冷启动池(200-1000)；知识类初始播放普遍低于娱乐，但收藏/转化更高、长尾更长；连续3条<200 需排查权重异常/违规。

### （2）1000-10000粉（成长期）
| 赛道 | 冷启动(1-3h) | 正常 | 优质 | 爆款门槛 |
|------|--------------|------|------|----------|
| 知识科普类(AI/科技) | 800-2000 | 3000-10000 | 1万-5万 | 10万+ |
| 泛娱乐类 | 1000-3000 | 5000-20000 | 2万-10万 | 20万+ |
| 生活技能类 | 900-2500 | 4000-15000 | 1.5万-8万 | 15万+ |

补充：粉丝贡献约20%-30%初始播放，冷启动通过率比新号高50%+；知识类千粉是起号黄金期。
- 「正常作品」=合规、垂直、无硬伤，达赛道均值。
- 「优质作品」=完播、收藏高于赛道均值30%+。

## 四、预测模型关键提示
1. 2026 升级 7 天长效赛马后，播放周期拉长，优质内容触发多次二次推流，最终可能远超24小时预测值，不能仅以首日定终局。
2. AI 生成内容合规双标注后进专属池获额外倾斜；**未标注内容预测时需乘 0.3-0.5 衰减系数**。
3. 知识类预测重点看**收藏率和复访率**，而非单纯点赞/播放，这两个决定长尾上限。
