# L2 违禁词库（平台 / 营销 / 知识专属）

> 整理日期：2026-06-24 ｜ 来源：`docs/sources/豆包_抖音规则手册.md` 第二部分（官方规范版）
> 检测覆盖：**文案 / 口播 / 字幕 / 标题 / 封面文字**。平台用 ASR 语音识别 + OCR 文字识别双重检测，**谐音 / 拼音 / 符号 / 变体（如 zui、最棒、wx）同样会被识别**，检测时一并考虑。

## 使用规则（重要）
1. **命中前必须先查 `白名单_L3.md`**：若该词落在白名单语境（如非营销语境的"最"、价格词"钱/便宜"），**不报或降级为🟡提示**，严禁误杀。
2. **语境分级**：营销 / 带货 / 推广语境从严（按下方严重度）；纯知识陈述语境，极限词可**降一档**（🔴→🟠）。
3. 每个命中项输出：`命中词 | 类别 | 严重度 | 安全替换`。

严重度：🔴 高（可能不过审 / 进不了冷启动）｜🟠 中（限流降权）｜🟡 低（建议优化）

## 一、广告法极限词（营销/带货 🔴；纯知识陈述 🟠）
> 溯源：《抖音电商广告禁用词规范》《广告法》

### 1. 最高级类 → 替换：主流 / 热门 / 行业前沿 / 个人推荐TOP / 我心中的优选
最、最佳、最优、顶级、史上第一、全网第一、销量第一、全国第一、NO.1、TOP1、销冠、榜首、独家、唯一、绝无仅有、前无古人、史无前例

### 2. 绝对化类 → 替换：大概率 / 多数人反馈 / 亲测有效 / 个人使用感受
100%有效、绝对、必定、包会、根治、永久、无敌、万能、零风险、无副作用、永不反弹

### 3. 权威资质类 → 替换：符合行业标准 / 通过相关检测 / 个人实测推荐
国家级、世界级、官方指定、国家认证、央视推荐、明星同款、驰名商标、质量免检

### 4. 诱导营销类 → 替换：限时活动 / 优惠进行中 / 活动期内福利
秒杀、抢疯了、再不买就没了、最后机会、清仓大甩卖、亏本甩卖、跳楼价、全网抄底价

## 二、知识类高频敏感词
> 溯源：《抖音创作者内容规范》

### 1. 虚假宣传类 🟠 → 替换：实用 / 提效明显 / 多数人觉得好用 / 一份可落地的方法
神器、黑科技、一夜暴富、躺赚、小白月入过万、零成本创业、无脑操作

### 2. 封建迷信类 🔴 → 建议删除，改为客观/科学表述
改运、招财、镇宅、辟邪、算命、看相、风水、运势、转运

### 3. 引流导流类 🔴（非企业蓝V账号 100% 违规）→ 删除，引导留在站内
微信、vx、wx、加群、私域、联系方式、二维码、主页找我、私信发你
（及暗示变体：懂的都懂、看我主页、粉丝群拿货 等）

### 4. 功效承诺类 🟠 → 替换：帮助你 / 更高效掌握 / 系统梳理（不承诺结果）
包过、保过、一次通关、7天速成、30天精通、学不会退款

## 三、AI 领域专属雷区 🔴
> 溯源：2026 抖音 AI 内容专项治理规则

1. 夸大替代类：宣称 AI "100%替代某职业 / 全面取代人工 / 让所有人失业" → 改为"辅助 / 提效 / 改变部分工作方式"
2. 违规用途类：宣传 AI 用于作弊 / 造假 / 侵权 / 绕过平台规则 → 删除
3. 虚构夸大类：虚构 AI 功能、夸大 AI 效果、声称 AI 能做到不存在的能力 → 改为客观描述实际能力
4. 名誉隐私类：用 AI 生成涉他人名誉 / 隐私的不实内容 → 删除
