# L3 白名单（防误杀）

> 整理日期：2026-06-24 ｜ 来源：`docs/sources/豆包_抖音规则手册.md` §2.4「常见误区澄清」
> **优先级：L3 高于 L2。** 任何词在 L2 命中前，先用本表判断；命中白名单语境则**不报**，避免误杀正常表达。

## 白名单条目

### 1. 价格描述词 —— 正常可用
钱、人民币、万、千元、便宜、贵
- 说明：属正常价格描述，**不是违禁词**，无需用"米、W"等谐音替代。
- 例：「这个工具一个月几十块钱」「便宜又好用」→ ✅ 不报。

### 2. 直播相关词 —— 正常可用
直播间、关注直播间
- 说明：可正常说，无需用"啵啵间、直间播"等不规范表述。

### 3. 「最」等极限词在非营销/非广告语境 —— 正常可用
- 说明：纯情感/客观陈述、非推广带货语境下，"最"等词不违规。
- 例：「我的母亲是世界上最伟大的人」→ ✅ 不报。
- 边界：一旦进入**营销/带货/推广产品**语境（如"全网最低价""效果最好的课"），仍按 L2 🔴 处理。

### 4. 提及其他平台名称 —— 正常可用
- 说明：可正常提及其他平台名称（如小红书、B站、公众号），**只要不刻意引导站外交易/导流**即可。
- 边界：若出现"去XX平台找我买""加XX群拿货"等导流意图 → 仍按 L2 引流类 🔴 处理。

---

## 判定流程（供 SKILL.md 调用）
```
对每个疑似命中词：
  if 命中 L3 白名单 且 当前语境符合白名单边界:
      → 不报（必要时在"白名单说明"里列出，告知用户"此词此处安全"）
  else:
      → 交给 L2 定类别 + 严重度 + 替换
```
关键：**先白名单、后违禁词**，宁可在报告里多一句"以下词此处安全"，也不要误杀让用户瞎改。
